Начало работы

Добро пожаловать в документацию AvenBot — вашего надёжного ИИ-ассистента для бизнеса!

AvenBot помогает автоматизировать процессы обработки сообщений, повысить эффективность общения с клиентами и улучшить качество обслуживания.

Наш сервис предлагает интеллектуальные решения для работы с популярными мессенджерами Telegram, WhatsApp и Max.

Быстрый старт

Перейдите на платформу AvenBot или установите приложение в YClients.

  1. Подключение к YClients
  2. Создание ассистента
  3. Настройка уведомлений
  4. Подключение каналов связи: Telegram, WhatsApp, Max
  5. Мониторинг чатов

Подключение к YClients

  1. Перейдите в маркетплейс YClients и найдите приложение AvenBot, либо откройте platform.avenbot.ru и нажмите кнопку "Подключить"

    Скриншот: yclients-step1.png (нужно предоставить)

  2. В появившемся окне нажмите "Продолжить"

    Скриншот: yclients-step2.png (нужно предоставить)

  3. Авторизуйтесь через Яндекс аккаунт

    Скриншот: yclients-step3.png (нужно предоставить)

  4. После авторизации нажмите кнопку для создания ассистента

    Скриншот: yclients-step4.png (нужно предоставить)

Создание ассистента

  1. Заполните название компании и выберите филиал

    Скриншот: assistant-step1.png (нужно предоставить)

  2. После создания ассистент появится в списке

  3. Дождитесь завершения инициализации ассистента

Настройка уведомлений

  1. Перейдите на вкладку "Правила" — здесь находятся базовые шаблоны уведомлений

  2. Настройте каскад уведомлений под ваши нужды

Подключение Telegram

  1. Перейдите в раздел "Каналы"

  2. Нажмите "Добавить канал" и выберите тип Telegram

  3. Создайте канал — если нет бот-токена, система предложит привязку через номер

    Скриншот: telegram-setup-step3.png (нужно предоставить)

  4. Отсканируйте QR-код для авторизации

    Скриншот: telegram-setup-step4.png (нужно предоставить)
Для расширенной настройки через бот-токен и Telegram Business смотрите раздел "Telegram бот + Business".

Подключение Max

  1. Перейдите в раздел "Каналы" и выберите тип Max

  2. Отсканируйте QR-код в приложении Max для авторизации

    Скриншот: max-step2.png (нужно предоставить)

Подключение WhatsApp

Подробная инструкция из 8 шагов по подключению WhatsApp.

  1. Перейдите на страницу управления ботами
  2. Выберите "Каналы"
  3. Нажмите кнопку "+ Добавить канал"
  4. Выберите "Тип" → WhatsApp и нажмите "Создать"
  5. Появится новый канал WhatsApp
  6. Нажмите на кнопку с QR-кодом
  7. Отсканируйте QR-код через приложение WhatsApp
  8. После успешного добавления отобразится статус подключения

Скриншоты для этого раздела уже есть в репозитории.

Мониторинг чатов

На вкладке "Чаты" вы можете отслеживать все диалоги и проверять доставку уведомлений.

Telegram бот + Business

Для корректной работы бота у вас должен быть подключён Telegram Premium.

Создание бота

  1. Перейдите в @BotFather
  2. Запустите мини-приложение и выберите добавление нового бота
  3. Заполните все поля, после нажмите "Создать бота"
  4. Включите Business mode в настройках бота
  5. Скопируйте токен бота

Подключение Telegram Business

  1. Добавьте новый канал связи на платформе
  2. Вставьте токен бота в поле настроек
  3. В Telegram перейдите в "Telegram Бизнес"
  4. Перейдите в "Чат боты"
  5. Добавьте ссылку на бота и включите все 1-2-1 сообщения

Скриншоты для этого раздела уже есть в репозитории.

Уведомления в Telegram

Для получения уведомлений о переводах на оператора необходимо настроить групповой чат с ботом.

Создание группы

  1. Откройте Telegram и создайте новую группу
  2. Добавьте в группу бота @ProdAvenAdminBot
  3. Отправьте команду /get-id
  4. Скопируйте ID чата (включая знак минус)
  5. Вставьте ID в настройках бота на платформе

Получение уведомлений

После настройки бот будет автоматически отправлять уведомления в групповой чат при переводе на оператора и напоминание через 24 часа.

Скриншоты для этого раздела уже есть в репозитории.
